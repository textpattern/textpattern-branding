h1. Textpattern branding

!https://david-dm.org/textpattern/textpattern-branding/dev-status.svg(devDependency Status)!:https://david-dm.org/textpattern/textpattern-branding#info=devDependencies

"Textpattern":http://textpattern.com branding resources.

h2. Install

You can include Textpattern branding resources in your project using "Bower":http://bower.io:

bc. $ bower install textpattern-branding

h2. Development

The repository uses "Grunt":http://gruntjs.com/ to run tasks. First make sure you have base dependencies installed: "Node.js":http://nodejs.org/ and "Grunt":http://gruntjs.com/. You can install Node using its installer and Grunt with npm:

bc. $ npm install -g grunt-cli

After you have the base dependencies taken care of, you can install the project's dependencies. Navigate to the project's directory, and run @npm@:

bc. $ cd textpattern-branding
$ npm install

After you have installed all dependencies, you will be able to run tasks using Grunt:

bc. $ grunt [task]

Where the @[task]@ is either @compress@ or @release[:patch | :minor | :major]@.
